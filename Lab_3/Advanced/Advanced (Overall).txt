Advanced:
* Скрипт, принимающий в качестве параметра командной строки число и выводящий на экран какую-нибудь строку указанное число раз:
index=0
while [[ $index < $1 ]]
do
echo Hello
let "index=index+1"
done

* Скрипт, суммирующий два числа, переданных в качестве параметров командной строки
let "p = $1 + $2"
echo $p
done

* Усовершенствование скрипта из предыдущего пункта так, чтобы в качестве третьего параметра можно было передавать желаемое арифметическое действие над числами:
#!/bin/bash
case $3 in
        add) let "p = $1 + $2";;
        sub) let "p = $1 - $2";;
        mul) let "p = $1 * $2";;
        div) let "p = $1 / $2";;
esac
echo $p

* Усовершенствование скрипта из предыдущего пункта так, чтобы при отсутствии третьего параметра он брал необходимую операцию из переменной окружения ARITHMETIC_OP:
#!/bin/bash
if [[ $3 == "" ]]
then
case $ARITHMETIC_OP in
        add) let "p = $1 + $2";;
        sub) let "p = $1 - $2";;
        mul) let "p = $1 * $2";;
        div) let "p = $1 / $2";;
esac
else
case $3 in
        add) let "p = $1 + $2";;
        sub) let "p = $1 - $2";;
        mul) let "p = $1 * $2";;
        div) let "p = $1 / $2";;
esac
fi
echo $p
