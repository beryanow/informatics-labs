Novice:
* Объясните, что делают команды ping и traceroute:
ping - диагностика сети, проверяет доступен ли удаленный сервер
traceroute - определение маршрута следования пакетов в сетях, работающих по межсетевому протоколу 
ping ya.ru / traceroute ya.ru
* IP-адрес у сервера yandex.ru:
ping yandex.ru -> 5.255.255.80

Advanced:
* Скрипт, принимающий в качестве параметра командной строки число и выводящий на экран какую-нибудь строку указанное число раз:
index=0
while [[ $index < $1 ]]
do
echo Hello
let "index=index+1"
done

* Скрипт, суммирующий два числа, переданных в качестве параметров командной строки
let "p = $1 + $2"
echo $p
done

* Усовершенствование скрипта из предыдущего пункта так, чтобы в качестве третьего параметра можно было передавать желаемое арифметическое действие над числами:
if [[ $3 == "add" ]]
then 
let "p = $1 + $2"
echo $p

else if [[ $3 == "sub" ]]
then
let "p = $1 - $2"
echo $p

else if [[ $3 == "mul" ]]
then
let "p = $1 * $2"
echo $p

fi
fi
fi
done

if [[ $3 == "" ]]

Expert:
* Найти слова, начинающиеся на ‘z’: egrep -o "\bz\w*\b" crusoe.txt
* Найти слова из 16 букв: egrep -o "\b\w{16}\b" crusoe.txt
* Найти слова, начинающиеся на 'a' и заканчивающиеся на ‘c’: egrep -o "\ba\w*c\b" crusoe.txt
* Найти слова, начинающиеся на 'ab', но у которых третья буква не ‘o’: egrep -o "\bab[^o]\w*\b" crusoe.txt
* Найти слова только из букв: egrep -w "\b[A-Z]*\b" patterns.txt
* Найти слова только из цифр: egrep -w "\b[0-9]*\b" patterns.txt
* Валидные российские автомобильные номера: egrep -w "\b[A,B,E,K,M,H,O,P,C,T,Y,X][0-9]{3}[A,B,E,K,M,H,O,P,C,T,Y,X]{2}[0-9]{2,3}\b" patterns.txt
